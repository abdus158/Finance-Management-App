# Design System

## Theme
Luminous Finance

## Style
- Dark mode first
- Glassmorphism
- Premium fintech aesthetic

## Colors

Background:
- `#0B1020`
- `#121826`

Primary Accent:
- `#00C9A7`

Danger:
- `#FF4D6D`

Card:
- `rgba(255,255,255,0.08)`

---

## Typography

Headings:
- Bold
- High contrast

Body:
- Medium weight
- Spacious line height

---

## UI Principles

- Minimal clutter
- Fast interaction
- High readability
- Data-first layout

---

## Animations

Allowed:
- Smooth transitions
- Graph animations
- Card expansion

Forbidden:
- Heavy unnecessary animations
- Slow motion effects
